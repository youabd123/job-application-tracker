# ?? Job Application Tracker

## ?? Projektbeskrivning
Detta projekt är en **konsolapplikation i C#** som hjälper användaren att hålla koll på sina jobbansökningar.  
Användaren kan:
- lägga till nya ansökningar  
- visa alla sparade ansökningar  
- uppdatera status på en ansökan  
- ta bort en ansökan  
- avsluta programmet  

Syftet med uppgiften är att träna på **objektorienterad programmering (OOP)**, **listor (List<T>)** och **menystrukturer** i C#.  

Projektet består av tre huvudfiler:
- `JobApplication.cs` – beskriver en enskild ansökan (företag, titel, status, lön, datum)
- `JobManager.cs` – hanterar listan med alla ansökningar (lägga till, visa, uppdatera, ta bort)
- `Program.cs` – innehåller menyn som användaren styr i konsolen
